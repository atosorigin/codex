codex.yodel.co.uk/products/[product name]

This API will give you all the information on a specific product, including its restrictions and product codes needed for integration with legacy DHL systems.

Call
GET http://codex.yodel.co.uk/products/[product name]?format=[xml,html]
Parameters
  • optional format
    Specified the responce type can take the value xml or html Note that is defaults to html format.
Responce

product
  • name - The name of the product e.g. EXPRESS-EXPRESS 24
  • product_code - e.g. 01
  • feature_code - e.g. 02
  • description - e.g. EXPRESSEXPRESS 24EXPRESS 24
  • handling_feature - e.g. STD
  • date_code - e.g. 0
  • day_text - e.g.
  • time_code - e.g. 0
  • time_text - e.g.
  • file_type - e.g. UKD

Example jquery code

This example list the product information for the EXPRESS-EXPRESS 24 product in a simple list

$.ajax({
  type: "GET",
  url: 'http://codex.yodel.co.uk/products/EXPRESS-EXPRESS 24?format=xml',
  dataType: "xml",
  success: function(data) {
    $('#result').empty();
    $('#result').append(
     "<ul id='product_info' ></ul>");
    $('#product_info').append(
     "<li>product name is " + $(data).find('name').text() + "</li>");
    $('#product_info').append(
     "<li>product code is " + $(data).find('product_code').text() + "</li>");
    $('#product_info').append(
     "<li>feature code is " + $(data).find('feature_code').text() + "</li>");
    $('#product_info').append(
     "<li>handeling feature is " + $(data).find('handling_feature').text() + "</li>");
    $('#product_info').append(
     "<li>description is " + $(data).find('description').text() + "</li>");
    })
  }
});
