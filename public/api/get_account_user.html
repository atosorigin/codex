GET codex.yodel.co.uk/accounts/[account name]/users/[username]

Gets the details for a specific user who belongs to an account. Please note that user name is only unique within an account.

Call
GET http://codex.yodel.co.uk/accounts/[account name]/users/[username]?format=[xml,html]
Parameters
  • optional format
    Specified the responce type can take the value xml or html Note that is defaults to html format.
Responce

user
  • account - the name of the account that the user belongs to e.g. acme
  • username - the user name for the user i.e. the string that they use to log in with.
  • email - the full email address of the user e.g. andy@foo.bar
  • role - the permission group for the user

Example Responce
<user>
  <account>acme</account>
  <username>admin</username>
  <email>geoff@acme.com</email>
<role>administrator</role>
Example jquery code

In this example we will get the details for the acme admin user and present them in a simple list

$.ajax({
  type: "GET",
  url: 'http://codex.yodel.co.uk/products/EXPRESS-EXPRESS 24?format=xml',
  dataType: "xml",
  success: function(data) {
    $('#result').empty();
    $('#result').append("<ul id='user' ></ul>");
    $('#user').append("<li>username=" + $(data).find('username').text() + "</li>");
    $('#user').append("<li>email=" + $(data).find('email').text() + "</li>");
    $('#user').append("<li>role=" + $(data).find('role').text() + "</li>");
  }
});
