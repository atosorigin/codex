codex.yodel.co.uk/products

The product API provides a list of all the YODEL products (i.e. different parcel delivery services) accross all of the postcodes that YODEL services.

Call
GET http://codex.yodel.co.uk/products?format=[xml,html]
Parameters
  • optional format
    Specified the responce type can take the value xml or html Note that is defaults to html format.
Responce
products
  • product
  • product
  • product
  • ...
Example Responce
<products>
  <product>EXPRESS-EXPRESS 24</product>
  <product>NATIONAL-EXPRESS 48</product>
</products>
Example jquery code

This simple example will get the list of YODEL products and proulate a list inside a list with the id "yodel_products"

$.ajax({
  type: "GET",
  url: 'http://codex.yodel.co.uk/products?format=xml',
  dataType: "xml",
  success: function(data) {
    $(data).find('product').each(function(){
      $('#yodel_products').append("<li>" + $(this).text() + "</li>");
    })
  }
});
