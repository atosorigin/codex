    What is the YODEL codex?

    The YODEL codex is a secure API that allows you to automatically place orders with YODEL, get labels for your parcels and track their delivery status. The API uses the ReST style so that you can integrate your business using the tools of your choice. This site provides resources, information and examples.

    To get information about our products or the status of your orders and parcels all you have to do is make a simple GET request to the YODEL codex which will return you the information in a structured format. To place orders or request labels you will need to POST information to the codex.

    If you are new to ReST or would like an interesting read we suggest that take a look at this article how I explained REST to my wife or read the excelent wikipedia artice on the subject.

    Getting Started with the YODEL codex

    The fastest way to get going with the codex is to try and use it. Lets try requesting a straight forward piece of information, the list of products that YODEL provides. You can do this by making a HTTP get request to this URL.

    http://codex.yodel.co.uk/products

    When you do this you will be returned the information on all the YODEL services in an XML format, It should look like this.

    <products>
      <product>
        <name>Lite</name>
        <description>
          Felxible delivery service for parcels that are Up to 2kg
          with dimensions up to 61x20x45cm
        </description>
      </product>
      ...
      <product>
        <name>Parcel</name>
        <description>
          Felxible delivery service for larger parcels that are up to 32kg with
          dimensions up to 90x60x60cm
        </description>
      </product>
    </products>

    This is a pretty simple example and while it provides great information on the YODEL services it its not much use when you want to send a parcel. Now lets look at another - much more usefull - example. When you are sending a parcel in the UK you will need to know which YODEL products you can use to get your parcel delivered. Imagine that you want to send a parcel to an address in the post code area SW17 8LW. The codex can help you in a very straight forward way. All you need to do is send a HTTP GET request to the collowing URL.

    http://codex.yodel.co.uk/postcodes/SW17 8LW/products

    Again the codex will return a porducts object in XML format, but this time it will only return the YODEL products that are in the SW18 8LW postcode region. You should get some XML back that looks like this.

    <products>
      <product>
        <name>Parcel</name>
        <description>
          Felxible delivery service for larger parcels that are up to 32kg with
          dimensions up to 90x60x60cm
        </description>
      </product>
    </products>

    We can see in this example that only our Mid Range product is available. Please note that we expand our products and services all the time, and that this is only an example.
